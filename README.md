# Library Management System using Linked List (C)

## Description
This project is a menu-driven Library Management System implemented in C using a singly linked list. It allows users to manage book records dynamically, including insertion, deletion, searching, sorting, and reversing the list.

## Features
- Insert new book records (Title, Author, Year)
- Delete a book by title
- Search for a book
- Update book details (Find and Replace)
- Count total number of books
- Sort books alphabetically by title
- Reverse the linked list
- Display all books

## Data Structure Used
- Singly Linked List

Each node contains:
- Title
- Author
- Year of publication
- Pointer to next node

## Program Structure
- `insert()` → Adds a new book to the list
- `delete()` → Removes a book by title
- `find()` → Searches for a book
- `findAndReplace()` → Updates book details
- `count()` → Returns total books
- `sort()` → Sorts books by title
- `reverse()` → Reverses the list
- `printList()` → Displays all records
